o objetivo do codigo é criar um API com arquitetura Restfull e  passar informações sobre filmes, como a nota que teve, o diretor que o dirigiu, os atores que estiveram no filme e a sua data de lançamento,
mostra tambem informações sobre os atores, quantos premios ganharam na vida(total), onde nasceu e quando  nasceu.

o que foi usado?
- Java
- Spring Boot
- Spring JPA
- Spring Security
- MySQL
- Validations
- Arquitetura em camadas
